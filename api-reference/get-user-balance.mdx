# GET /api/getUserBalance

Get the current balance of a user. This endpoint returns the available balance in the user's wallet.

## Request

<RequestExample>
```bash cURL
curl -X GET "https://api.braix.com.br/api/getUserBalance" \
  -H "Authorization: Bearer your_access_token"
```
</RequestExample>

<CodeGroup>
```javascript JavaScript
const axios = require('axios');

const getUserBalance = async () => {
  const response = await axios.get('https://api.braix.com.br/api/getUserBalance', {
    headers: {
      'Authorization': `Bearer ${process.env.BRAIX_ACCESS_TOKEN}`
    }
  });
  
  return response.data;
};
```

```python Python
import requests

def get_user_balance():
    headers = {
        'Authorization': f'Bearer {os.getenv("BRAIX_ACCESS_TOKEN")}'
    }
    
    response = requests.get('https://api.braix.com.br/api/getUserBalance', headers=headers)
    return response.json()
```

```php PHP
<?php
function getUserBalance() {
    $token = $_ENV['BRAIX_ACCESS_TOKEN'];
    
    $context = stream_context_create([
        'http' => [
            'header' => [
                'Authorization: Bearer ' . $token
            ],
            'method' => 'GET'
        ]
    ]);
    
    $response = file_get_contents('https://api.braix.com.br/api/getUserBalance', false, $context);
    return json_decode($response, true);
}
?>
```
</CodeGroup>

## Response Fields

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="balance" type="number">
  The current available balance in BRL.
</ResponseField>

<ResponseField name="currency" type="string">
  The currency code (always "BRL" for Brazilian Real).
</ResponseField>

<ResponseField name="lastUpdated" type="string">
  The timestamp when the balance was last updated.
</ResponseField>

<ResponseField name="wallet" type="object">
  Wallet information including ID and status.
</ResponseField>
