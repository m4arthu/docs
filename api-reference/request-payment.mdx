# POST /requestPayment

<Badge variant="success">POST</Badge>

Send a direct PIX payment to a specific PIX key. This endpoint allows you to transfer money directly to a recipient's PIX key.

## Request

<RequestExample>
```bash cURL
curl -X POST "https://api.braix.com.br/requestPayment" \
  -H "Authorization: Bearer your_access_token" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10.50,
    "pixKey": "user@example.com",
    "webhookUrl": "https://your-domain.com/webhooks/braix",
    "externalId": "order_123",
    "description": "Payment for order #123",
    "withdrawalType": "qrCode"
  }'
```
</RequestExample>

<CodeGroup>

```python Python
import requests

def send_payment(amount, pix_key, webhook_url, external_id, description, withdrawal_type):
    headers = {
        'Authorization': f'Bearer {os.getenv("BRAIX_ACCESS_TOKEN")}',
        'Content-Type': 'application/json'
    }
    
    data = {
        'amount': amount,
        'pixKey': pix_key,
        'webhookUrl': webhook_url,
        'externalId': external_id,
        'description': description,
        'withdrawalType': withdrawal_type
    }
    
    response = requests.post('https://api.braix.com.br/requestPayment', 
                           json=data, headers=headers)
    return response.json()
```

```php PHP
<?php
function sendPayment($amount, $pixKey, $webhookUrl, $externalId, $description, $withdrawalType) {
    $token = $_ENV['BRAIX_ACCESS_TOKEN'];
    $data = [
        'amount' => $amount,
        'pixKey' => $pixKey,
        'webhookUrl' => $webhookUrl,
        'externalId' => $externalId,
        'description' => $description,
        'withdrawalType' => $withdrawalType
    ];
    
    $context = stream_context_create([
        'http' => [
            'header' => [
                'Authorization: Bearer ' . $token,
                'Content-Type: application/json'
            ],
            'method' => 'POST',
            'content' => json_encode($data)
        ]
    ]);
    
    $response = file_get_contents('https://api.braix.com.br/requestPayment', false, $context);
    return json_decode($response, true);
}
?>
```
</CodeGroup>

## Request Body

<ParamField query="amount" type="number" required>
  The payment amount in BRL.
</ParamField>

<ParamField query="pixKey" type="string" required>
  The recipient's PIX key (email, CPF, phone, or random key).
</ParamField>

<ParamField query="webhookUrl" type="string" required>
  The webhook URL to receive payment notifications.
</ParamField>

<ParamField query="externalId" type="string">
  External identifier for the transaction.
</ParamField>


<ParamField query="description" type="string">
  Description of the payment.
</ParamField>

<ParamField query="withdrawalType" type="string">
  Type of withdrawal. Options: qrCode, crypto.
</ParamField>
