# Welcome to PinePay API

PinePay is a comprehensive PIX payment processing API that provides seamless integration with multiple payment gateways in Brazil. Our API enables you to accept PIX payments, process refunds, handle webhooks, and manage transactions with enterprise-grade reliability.

## What is PinePay?

PinePay is a unified API that abstracts the complexity of multiple PIX payment gateways, providing you with a single integration point for all your payment processing needs. Whether you're building an e-commerce platform, SaaS application, or fintech solution, PinePay simplifies PIX payment integration.

## Supported Gateways

PinePay supports integration with the following payment gateways:

- **Woovi** - OpenPix integration for instant PIX payments
- **Treeal** - Advanced PIX processing with fraud detection
- **Voluti** - High-performance PIX gateway
- **EFI** - Efi Bank PIX integration
- **PlayShark** - Gaming-focused payment processing
- **Pluggou** - Modern PIX payment solutions
- **BancoE2** - Banking-grade PIX processing
- **PrimePag** - E-commerce PIX solutions
- **WePayments** - Multi-channel payment processing
- **XGate** - Enterprise PIX gateway

## Architecture Overview

<Frame>
<img src="/images/architecture.png" alt="PinePay API Architecture showing request flow through gateways" />
</Frame>

PinePay API follows a microservices architecture with the following components:

- **API Gateway** - Main API server handling requests and authentication
- **Queue System** - RabbitMQ-based background processing
- **Database Layer** - MySQL/PostgreSQL for transaction storage
- **Cache Layer** - Redis for high-performance caching
- **Monitoring** - Sentry integration for error tracking and performance monitoring
