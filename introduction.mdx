# Welcome to braix API

braix is a comprehensive PIX payment processing API that provides seamless integration with multiple payment gateways in Brazil. Our API enables you to accept PIX payments, process refunds, handle webhooks, and manage transactions with enterprise-grade reliability.

## What is braix?

braix is a unified API that abstracts the complexity of multiple PIX payment gateways, providing you with a single integration point for all your payment processing needs. Whether you're building an e-commerce platform, SaaS application, or fintech solution, braix simplifies PIX payment integration.

## Getting Started

Ready to start integrating PIX payments? Here's what you need to know:

<Steps>
<Step title="Get API Credentials">
  Sign up for a braix account and obtain your API key from the dashboard.
</Step>


<Step title="Make Your First Request">
  Use our simple REST API to create your first PIX payment.
</Step>

<Step title="Handle Webhooks">
  Set up webhook endpoints to receive real-time payment notifications.
</Step>
</Steps>
