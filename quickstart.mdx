# Quick Start Guide

This guide will walk you through setting up braix API integration in just a few steps. You'll be processing PIX payments in no time!

## Prerequisites

Before you begin, make sure you have:

- A braix account ([Sign up here](https://dashboard.braix.com.br))
- Your API credentials from the dashboard
- Node.js 16+ or your preferred programming language
- A webhook endpoint URL for receiving notifications

## Step 1: Install Dependencies

<CodeGroup>
```bash npm
npm install axios
```

```bash yarn
yarn add axios
```

```bash pip
pip install requests
```

```bash composer
composer require guzzlehttp/guzzle
```
</CodeGroup>

## Step 2: Configure Your Environment

Create a `.env` file with your braix credentials:

```bash .env
braix_API_KEY=your_api_key_here
braix_BASE_URL=https://api.braix.com
braix_WEBHOOK_URL=https://your-domain.com/webhooks/braix
```

<Warning>
Never commit your API keys to version control. Use environment variables or secure configuration management.
</Warning>

## Step 3: Make Your First API Call

Let's create a simple PIX payment request:

<CodeGroup>
```javascript Node.js
const axios = require('axios');

const createPixPayment = async () => {
  try {
    const response = await axios.post('https://api.braix.com/requestPayment', {
      amount: 10.50,
      pixKey: 'user@example.com',
      webhookUrl: 'https://your-domain.com/webhooks/braix',
      externalId: 'order_123',
      gateway: 'woovi',
      description: 'Payment for order #123'
    }, {
      headers: {
        'Authorization': `Bearer ${process.env.braix_API_KEY}`,
        'Content-Type': 'application/json'
      }
    });

    console.log('Payment created:', response.data);
    return response.data;
  } catch (error) {
    console.error('Error creating payment:', error.response?.data || error.message);
  }
};

createPixPayment();
```

```python Python
import requests
import os

def create_pix_payment():
    url = 'https://api.braix.com/requestPayment'
    headers = {
        'Authorization': f'Bearer {os.getenv("braix_API_KEY")}',
        'Content-Type': 'application/json'
    }
    data = {
        'amount': 10.50,
        'pixKey': 'user@example.com',
        'webhookUrl': 'https://your-domain.com/webhooks/braix',
        'externalId': 'order_123',
        'gateway': 'woovi',
        'description': 'Payment for order #123'
    }
    
    try:
        response = requests.post(url, json=data, headers=headers)
        response.raise_for_status()
        print('Payment created:', response.json())
        return response.json()
    except requests.exceptions.RequestException as e:
        print('Error creating payment:', e)
        return None

create_pix_payment()
```

```php PHP
<?php
$apiKey = $_ENV['braix_API_KEY'];
$url = 'https://api.braix.com/requestPayment';

$data = [
    'amount' => 10.50,
    'pixKey' => 'user@example.com',
    'webhookUrl' => 'https://your-domain.com/webhooks/braix',
    'externalId' => 'order_123',
    'gateway' => 'woovi',
    'description' => 'Payment for order #123'
];

$options = [
    'http' => [
        'header' => [
            'Authorization: Bearer ' . $apiKey,
            'Content-Type: application/json'
        ],
        'method' => 'POST',
        'content' => json_encode($data)
    ]
];

$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);

if ($result === FALSE) {
    echo 'Error creating payment';
} else {
    echo 'Payment created: ' . $result;
}
?>
```
</CodeGroup>

## Step 4: Handle Webhooks

Set up a webhook endpoint to receive payment notifications:

<CodeGroup>
```javascript Node.js
const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhooks/braix', (req, res) => {
  const { type, data } = req.body;
  
  if (type === 'PAYMENT_RECEIVED') {
    console.log('Payment received:', data);
    // Update your database, send confirmation email, etc.
  }
  
  res.status(200).json({ received: true });
});

app.listen(3000, () => {
  console.log('Webhook server running on port 3000');
});
```

```python Python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/webhooks/braix', methods=['POST'])
def handle_webhook():
    data = request.get_json()
    webhook_type = data.get('type')
    
    if webhook_type == 'PAYMENT_RECEIVED':
        print('Payment received:', data.get('data'))
        # Update your database, send confirmation email, etc.
    
    return jsonify({'received': True})

if __name__ == '__main__':
    app.run(port=3000)
```

```php PHP
<?php
// webhook.php
$input = file_get_contents('php://input');
$data = json_decode($input, true);

$type = $data['type'] ?? null;

if ($type === 'PAYMENT_RECEIVED') {
    error_log('Payment received: ' . json_encode($data['data']));
    // Update your database, send confirmation email, etc.
}

http_response_code(200);
echo json_encode(['received' => true]);
?>
```
</CodeGroup>

<AccordionGroup>
<Accordion title="Common Issues">
  - **Authentication errors**: Verify your API key is correct and has proper permissions
  - **Webhook not received**: Ensure your endpoint is publicly accessible and returns 200 status
  - **Payment not processed**: Check if the gateway is active and properly configured
</Accordion>

  - Always validate webhook signatures in production
  - Implement proper error handling and retry logic
