# Quick Start Guide

This guide will walk you through setting up braix API integration in just a few steps. You'll be processing PIX payments in no time!

## Prerequisites

Before you begin, make sure you have:

- A braix account ([Sign up here](https://dashboard.braix.com.br))
- Your API credentials from the dashboard
- Node.js 16+ or your preferred programming language
- A webhook endpoint URL for receiving notifications

## Step 1: Install Dependencies

<CodeGroup>
```bash npm
npm install axios
```

```bash yarn
yarn add axios
```

```bash pip
pip install requests
```

```bash composer
composer require guzzlehttp/guzzle
```
</CodeGroup>

## Step 2: Configure Your Environment

Create a `.env` file with your braix credentials:

```bash .env
BRAIX_CLIENT_ID=your_client_id_here
BRAIX_CLIENT_SECRET=your_client_secret_here
BRAIX_BASE_URL=https://api.braix.com.br
BRAIX_WEBHOOK_URL=https://your-domain.com/webhooks/braix
```

<Warning>
Never commit your API keys to version control. Use environment variables or secure configuration management.
</Warning>

## Step 3: Get Access Token

First, you need to obtain an access token using your client credentials:

<CodeGroup>
```javascript Node.js
const axios = require('axios');

const getAccessToken = async () => {
  const clientId = process.env.BRAIX_CLIENT_ID;
  const clientSecret = process.env.BRAIX_CLIENT_SECRET;
  const credentials = Buffer.from(`${clientId}:${clientSecret}`).toString('base64');
  
  const response = await axios.post('https://api.braix.com.br/auth/token', {}, {
    headers: {
      'Authorization': `Basic ${credentials}`,
      'Content-Type': 'application/json'
    }
  });
  
  return response.data.access_token;
};
```

```python Python
import requests
import base64
import os

def get_access_token():
    client_id = os.getenv('BRAIX_CLIENT_ID')
    client_secret = os.getenv('BRAIX_CLIENT_SECRET')
    credentials = base64.b64encode(f"{client_id}:{client_secret}".encode()).decode()
    
    response = requests.post('https://api.braix.com.br/auth/token', 
                           headers={'Authorization': f'Basic {credentials}'})
    return response.json()['access_token']
```

```php PHP
<?php
function getAccessToken() {
    $clientId = $_ENV['BRAIX_CLIENT_ID'];
    $clientSecret = $_ENV['BRAIX_CLIENT_SECRET'];
    $credentials = base64_encode($clientId . ':' . $clientSecret);
    
    $context = stream_context_create([
        'http' => [
            'header' => [
                'Authorization: Basic ' . $credentials,
                'Content-Type: application/json'
            ],
            'method' => 'POST'
        ]
    ]);
    
    $response = file_get_contents('https://api.braix.com.br/auth/token', false, $context);
    $data = json_decode($response, true);
    return $data['access_token'];
}
?>
```
</CodeGroup>

## Step 4: Make Your First API Call

Now let's create a PIX QR Code using the access token:

<CodeGroup>
```javascript Node.js
const axios = require('axios');

const createPixQrCode = async () => {
  try {
    // First get the access token
    const token = await getAccessToken();
    
    // Then make the API call with Bearer token
    const response = await axios.post('https://api.braix.com.br/requestQrCode', {
      amount: 10.50,
      webhookurl: 'https://your-domain.com/webhooks/braix'
    }, {
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json'
      }
    });

    console.log('QR Code created:', response.data);
    return response.data;
  } catch (error) {
    console.error('Error creating QR Code:', error.response?.data || error.message);
  }
};

createPixQrCode();
```

```python Python
import requests
import base64
import os

def create_pix_qr_code():
    # First get the access token
    token = get_access_token()
    
    # Then make the API call with Bearer token
    headers = {
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    }
    
    data = {
        'amount': 10.50,
        'webhookurl': 'https://your-domain.com/webhooks/braix'
    }
    
    try:
        response = requests.post('https://api.braix.com.br/requestQrCode', 
                               json=data, headers=headers)
        response.raise_for_status()
        print('QR Code created:', response.json())
        return response.json()
    except requests.exceptions.RequestException as e:
        print('Error creating QR Code:', e)
        return None

create_pix_qr_code()
```

```php PHP
<?php
function createPixQrCode() {
    // First get the access token
    $token = getAccessToken();
    
    // Then make the API call with Bearer token
    $data = [
        'amount' => 10.50,
        'webhookurl' => 'https://your-domain.com/webhooks/braix'
    ];
    
    $context = stream_context_create([
        'http' => [
            'header' => [
                'Authorization: Bearer ' . $token,
                'Content-Type: application/json'
            ],
            'method' => 'POST',
            'content' => json_encode($data)
        ]
    ]);
    
    $response = file_get_contents('https://api.braix.com.br/requestQrCode', false, $context);
    
    if ($response === FALSE) {
        echo 'Error creating QR Code';
    } else {
        echo 'QR Code created: ' . $response;
    }
}

createPixQrCode();
?>
```
</CodeGroup>

## Step 5: Handle Webhooks

Set up a webhook endpoint to receive PIX payment notifications. braix sends webhooks for the following events:

- **RECEIVE**: PIX deposit confirmed and liquidated
- **TRANSFER**: PIX withdrawal confirmed and liquidated  
- **CASHOUT**: PIX withdrawal rejected or cancelled
- **REFUND**: Transaction refunded (deposit or withdrawal)
- **INFRACTION**: PIX infraction reported or closed

<CodeGroup>
```javascript Node.js
const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhooks/braix', (req, res) => {
  const { type, data } = req.body;
  
  if (type === 'RECEIVE') {
    console.log('PIX deposit received:', data);
    // Update your database, send confirmation email, etc.
  }
  
  if (type === 'TRANSFER') {
    console.log('PIX withdrawal completed:', data);
    // Handle withdrawal completion
  }
  
  if (type === 'CASHOUT') {
    console.log('PIX withdrawal rejected:', data);
    // Handle withdrawal rejection
  }
  
  if (type === 'REFUND') {
    console.log('Transaction refunded:', data);
    // Handle refund
  }
  
  if (type === 'INFRACTION') {
    console.log('PIX infraction:', data);
    // Handle infraction
  }
  
  res.status(200).json({ received: true });
});

app.listen(3000, () => {
  console.log('Webhook server running on port 3000');
});
```
