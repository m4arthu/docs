# Changelog

Todas as mudanças notáveis nesta documentação serão documentadas neste arquivo.

## [1.0.0] - 2025-01-19

### Adicionado
- Documentação completa da API braix
- Guias de integração PIX
- Documentação de webhooks com seção dedicada
- Exemplos de código em Node.js, Python e PHP
- Validação de segurança por checksum
- Guias de troubleshooting

### Alterado
- Reorganizada estrutura de navegação
- Traduzida documentação para português
- Atualizada validação de webhooks para usar apenas checksum

### Removido
- Validação de assinatura X-braix-Signature (substituída por checksum)

## [0.9.0] - 2025-01-18

### Adicionado
- Estrutura inicial da documentação
- Páginas de API Reference
- Guias básicos de integração
